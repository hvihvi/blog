# Hugo Vinet

Notes and articles on subjects mostly related to front-end engineering.

Built with **Next.js**, a library called [Nextra](https://nextra.vercel.app/) and deployed with [Vercel](https://vercel.com/). It allows you to write Markdown, deploy with `git push` and focus on the _content_.
